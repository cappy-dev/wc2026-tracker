# Contributing to WC 2026 Tracker

Thanks for your interest in contributing! This is a static site (HTML, CSS, vanilla JS) that pulls live World Cup data from the ESPN public API.

## Quick Start

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/wc2026-tracker.git`
3. Open `index.html` in a browser — no build step or server required

## Making Changes

- Keep it simple: no frameworks, no build tools, no API keys
- Test changes by opening the relevant HTML file directly in a browser
- Follow the existing code style (2-space indent in JS/CSS)
- Make sure the dark/light theme toggle still works after UI changes

## Pull Requests

- One focused improvement per PR (e.g., a bug fix, a new feature, a docs update)
- Include a clear description of what the PR does and why
- If fixing a visual issue, mention which page(s) are affected (index, standings, matches, scorers, teams)

## Reporting Issues

Found a bug or have an idea? Open an issue with:

- Which page it occurs on
- Steps to reproduce
- What you expected vs. what happened
- Browser and device info (if relevant)

## Ideas for Contributions

- Accessibility improvements (ARIA labels, keyboard navigation)
- Mobile responsiveness fixes
- Additional ESPN API data (lineups, events timeline)
- Better error states or loading indicators
- Translations / i18n support
